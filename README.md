# 💳 Credit Card Fraud Detection System

## 📌 Project Overview

The **Credit Card Fraud Detection System** is a machine learning–based web application designed to detect fraudulent credit card transactions. The system analyzes transaction data and predicts whether a transaction is **fraudulent** or **legitimate** using classification algorithms.

This project demonstrates practical skills in **Python, Machine Learning, Data Analysis, and Web Development**, and can be used as a portfolio project for internships and entry-level software developer roles.

---

## 🎯 Objectives

* Detect fraudulent credit card transactions using machine learning models
* Improve transaction security by identifying suspicious activities
* Provide a simple web interface for prediction
* Demonstrate real-world application of data science and Python development

## ⚙️ Installation Steps

Follow these steps to run the project locally:

1. **Clone the repository**

```
git clone https://github.com/your-username/Credit-card-Fraud-Detection.git
```

2. **Navigate to the project folder**

```
cd Credit-card-Fraud-Detection
```

3. **Install dependencies**

```
pip install -r requirements.txt
```

4. **Run the application**

```
python app.py
```

5. **Open in browser**

```
http://127.0.0.1:5000/
```
